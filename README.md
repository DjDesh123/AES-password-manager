AES-password-manager
a secure password manager written with flask and AES encryption.


Description:
This project is a lightweight password manager that encrypts stored passwords using AES encryption, the reason for this is to provide data security and privacy.The flask based UI to provide users with an easy and user friendly experience.


Features:
AES-256 encryption for user passwords
Flask based web based ui
Secure login and authentication system
back up and restore functionality
